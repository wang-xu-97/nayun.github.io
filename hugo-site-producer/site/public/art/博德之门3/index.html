经典combo
#

  1. 火（引爆）
    油脂瓶/油腻术、油桶、酒桶、烟粉桶、烟粉挎包、烟粉炸弹
  2. 水（濡湿）
    1. 生成：
      造水术/投掷水瓶/天然地形/融化冰面
    2. 闪电易伤：
      巫术箭闪电伤害
    3. 冰冻
      悼霜冰冻跳回合+攻击优势
    4. 形成冰面
      概率滑倒、减移速
  3. 水电工
    1. build：
      1. 核心：副手手弩 –> 附动射水瓶造濡湿
      2. 3术点出超魔、2牧出引导神力
      3. 蝌蚪异界强运
    2. 手法
      1. 通用：附动造濡湿（闪电冰冻易伤）
      2. 超魔成双冷冻射线：
        1. 法术环位全转为术法点
        2. 成双冷冻射线
      3. 异界强运+引导神力
        1. 打单：巫术箭
        2. 群伤：闪电束
      4. 吃嗜血灵药+速度油打召雷术

经典buff
#

  1. 伤害
    投掷之戒：投掷1d4
    星界银剑：心灵附伤1d6
    蘸火：1d4火焰伤害
    元素战刃：1d6指定元素伤害
    法术火花：法术/戏法命中增加闪电充能
  2. 命中
    轻声承诺：1d4命中+1d4豁免 2R
  3. npc检定
    1. 交友术：魅力属性检定优势骰（2次d20取高）
      结束时，npc好感降低、可能敌对，处理方法：结束前离开npc视野范围
    2. 神导术：1d4任意属性检定掷骰

技巧
#

1. 战斗前准备
#

  1. 开buff
    1. 移动：
      1. 卡疾走buff 1R
      2. 开大跳/羽落 10R
    2. 增伤：
      1. 生死离别 5R
    3. 命中：
      1. 祝福术 10R
      2. 喝药+轻声承诺 2R
    4. 其他：
      1. 加速术 10R，加速药 3R
  2. 布置场地
    1. 投掷药水，持续6s，约1回合
      1. 把药水投掷到预定位置，任何人踩上去直接生效
        1. 踩回复药水 + 轻声承诺 命中buff 2R
    2. 摆放水瓶/油脂瓶，预判敌人位置，（aoe/附动攻击/环境）打破上濡湿(3R)/油脂(10R)
    3. 投掷油脂瓶/油腻术 油脂地面 10R，投掷水瓶/造水术 水面 3R
    4. 使用炼金火焰/油脂瓶+火焰箭制造火焰地表 2R
    5. 水面+火 形成蒸汽云（只能被驱散云雾法术驱散），无限濡湿、免疫燃烧、轻度遮蔽
      闪电命中形成带电蒸汽云1d4闪电伤害，2d8闪电易伤
    6. 造水术+冰（冰刃/守卫刻文冰冻）造大范围冰面
      1. 使用守卫刻文可以节省冰面回合数，因为敌人踩上来才生成，并且可能滑倒无法动作

2. 战斗中无消耗动作
#

  1. 物品分割、世界位置移动
    1. 手法：
      1. 近距离，通过人物自己的范围移动物品
      2. 远距离，通过处在行动回合的其他队友/法师之手的范围移动物品
    2. 技巧收益思路：
      1. 利用环境伤害破坏物品
    3. 应用场景
      1. 虚空球茎被移动到火焰地表炸裂
        1. 可以打破水瓶无消耗制造濡湿（火焰地表 -> 虚空球茎爆裂 -> 打破水瓶 -> 濡湿）

3. 临时武器攻击
#

  1. 非战斗状态把中立npc搬走，控制入战位置/偷窃npc视野
  2. 战斗中投掷敌人（扔下深渊秒杀，无法搜刮尸体），需力量检定
    常见判定：
    轻物（≤50磅）：自动成功。
    中等（50~200磅）：DC 10~15力量检定。
    重物（≥200磅）：DC 20+，或需多人协作。

4. 偷窃
#

  1. 黑暗箭创造黑暗遮蔽区域
  2. 临时武器攻击搬开npc，转移视野
  3. 偷钱/堆叠物品，先分割物品，小份物品偷窃检定值更低
  4. 瓦罗可以无限偷，即使被发现
  5. 区分+巧手的偷窃专用套装和战斗套装

5. 具体场景
#

1. 第一章星界银剑
#

准备：

  • 药水：1个速度油、2个隐身药水
  • 法术：自我伪装、命令术/缴械攻击远程、云雾术
  • 人物：2人，单个控制
    1. A: 带1速度油1隐身药，自我伪装、命令术/缴械攻击远程
    2. B: 隐身药/云雾术
      云雾术谁带都行，一个带云雾术，另一个带隐身药

操作：

  1. B：站在木桥路牌前，回营地
  2. A:
    1. 自我伪装后站在楼上，向指挥官扔速度油（溅射，不能打到）
    2. 等18s指挥官昏睡后用命令术/缴械攻击打掉武器，失败读档
    3. 喝隐身药
    4. 逃跑脱离战斗
    5. 回营地
  3. 长休 (如果出营地发现养鸡人还在楼上，不然可以不用长休)
  4. 带云雾术的人上楼，在视野外用云雾术盖住星界银剑
  5. 另一人喝隐身药走进云雾术拾取星界银剑，双人传送脱战

规则
#

基础负重规则
#

角色负重能力 = 力量值 × 15（磅）

npc
#

  1. 可以卖给商人背包，帮商人把杂物都装到背包里，整理商人物品

专注豁免
#

  1. 受到伤害，为每个伤害源过一次体质豁免检定(1d20+体质调整值)，DC为max(10, 当次伤害值/2)
  2. 直接失去专注的情况：倒地、睡眠、愤怒、失能

机制
#

攻击优势（造优）
#

  1. 高地
  2. 敌人受惊
    在敌人视野外攻击、并被发现
  3. 敌人倒地
    1. 命令术：匍匐
    2. 冰面、油脂地面
  4. buff
    1. （人类/怪物）定身术
    2. 妖火
    3. 曳光弹
    4. 碎魂者

状态
#

  1. 濡湿
  2. 冰冻
    濡湿状态的生物在被施加冻僵状态时会转变为冰冻状态，冰冻状态下失能一回合并对钝击、雷鸣、力场三种伤害有一次性易伤，但寒霜易伤和闪电易伤会被覆盖。

环境反应
#

1. 水
#

  1. 冰面
    1. 水/血液都可以结冰，在冰面上移动会被视为劣势地形有移速惩罚，而且有概率滑倒跳过一回合
    2. 生成：
      1. 雪花迸射戒指
      2. 结冰的条件只需要对水面造成任何寒冰伤害即可，被链接的所有水面都会被结为冰块，结冰时不会造成任何影响只有在开始移动时才会有影响，冰面会再两回合后融化
  2. 带电的水
    当水面被闪电法术影响时还会转变为带电的水，每回合对上面的所有生物造成1d4的伤害。
  3. 蒸汽云
    水与火焰接触后会形成蒸汽云，只要处于蒸汽云内就会被无限施加濡湿状态，并且免疫燃烧以及处于遮蔽状态，不会对内外的攻击施加劣势与优势。
    蒸汽云不会被任何新地表覆盖与驱散，只有那几个专门写了驱逐云雾的法术才能将其驱散，并且与水类似的是在被闪电法术影响之后也会不断造成1d4的闪电伤害，但永续的易伤实际上是2d8

2. 火
#

  1. 火焰地表:
    1. 扔炼金火焰
    2. 油脂地面(油脂瓶/油脂术/油桶)、酒精地面(酒桶)+火焰法术
