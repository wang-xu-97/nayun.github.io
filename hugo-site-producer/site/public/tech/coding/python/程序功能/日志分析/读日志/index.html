 1def readlog(logfile, st, et):

3. 反向读日志
#

思路：定位到文件尾，逐行读取
核心：

  1. 定位文件尾：
    file.seek(0, 2)
  2. 逐个字符读取，直到换行符yield输出：
    1. buffer = file.read(1) + buffer 需要捕获处理file.read的异常
    2. if buffer[0] == '\n':yield buffer
</> python 
 1def r_read(fp):
 2    with open(fp, 'rb') as f:
 3        f.seek(0, 2)
 4        pos = f.tell()
 5        buffer = ''
 6        while pos >= 0:
 7            f.seek(pos)
 8            try:
 9                c = f.read(1)
10            except:
11                print('decode error, pass')
12                c = ''
13            if c == '\n':
14                yield buffer
15                buffer = ''
16            else:
17                buffer = c + buffer
18            pos -= 1
19        # 边界情况
20        yield buffer
