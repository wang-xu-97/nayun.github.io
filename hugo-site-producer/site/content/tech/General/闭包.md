在创建时封装当时状态，并使当时的状态生存周期延长至与闭包生存周期一致
闭包思想产物：
	1. python装饰器
	2. 回调函数
	3. 函数工厂
	4. 惰性求值
	...

# 典型用法
## 1. python 装饰器
```python
def execute(f):
	f()

@execute
def execute_when_defined():
	print('executed')

```
