# 0. 专长

| 专长 | 效果描述 | 推荐度与备注 |
| :--- | :--- | :--- |
| **属性提升** | 一项属性+2，或两项属性各+1（上限20） | 通用选择，缺啥补啥 |
| **演员** | 魅力+1（上限20），表演和欺瞒检定熟练加值翻倍 | 适合魅力型角色，剧情流可选 |
| **警觉** | **先攻+5**，不會陷入受驚狀態 | **極力推薦**，几乎所有角色都受益于高先攻 |
| **運動員** | 力量或敏捷+1（上限20），倒地站立所需移動力更低，**跳躍距離增加50%** | 近战角色不错，跳跃是强大的机动手段 |
| **衝鋒** | 獲得衝擊：武器攻擊以及衝擊：推擊 | 不推荐，动作消耗大，性价比低 |
| **強弩專家** | 近戰用弩無劣勢，穿刺射擊開放傷口時長翻倍 | 弩箭手可选，应对近身贴脸 |
| **防禦式決鬥** | 被熟練的靈巧武器近戰命中時，可用反應使AC+2 | 适合敏捷型近战角色，相当于无限次“护盾术” |
| **雙持客** | **可雙持非輕型武器**，雙持近戰武器時AC+1 | **法師必學**（双持法杖），近战双持流可选 |
| **地牢探索者** | 察覺偵測隱藏、豁免抵抗陷阱有優勢，有陷阱傷害抗性 | 不推荐，宠物或队友可处理陷阱 |
| **歷久彌堅** | 體質+1（上限20），**短休恢復全部生命值** | 前期开荒可选，后期药水多，作用下降 |
| **元素導師** | 選擇一種法術類型無視其傷害抗性，對應法術不會骰出1 | 法系输出可选，但若打易伤则收益低 |
| **巨武器大師** | **重擊或擊殺後可用附贈動作再攻擊一次**；可選-5命中+10傷害 | **雙手武器近戰必學**，需配合优势或命中加成抵消惩罚 |
| **重甲運用** | 獲得重甲熟練項，力量+1（上限20） | 需要重甲但职业不提供时才考虑，通常职业自带 |
| **重甲大師** | 力量+1（上限20），穿重甲時非魔法攻擊的繼發傷害-3 | **重甲坦克強力推薦**，显著提升生存能力 |
| **輕甲運用** | 獲得輕甲熟練項，力量或敏捷+1（上限20） | 需要轻甲但职业不提供时才考虑 |
| **幸運** | 獲得3點幸運點（長休回復），可用於讓自身檢定獲優勢或讓敵人重骰攻擊 | **泛用性強**，戰鬥内外都有用 |
| **法師殺手** | 近戰範圍內敵人施法時，其豁免檢定處於優勢，可反應攻擊，命中後敵人專注檢定劣勢 | PVP強，PVE法師怪較少威脅一般 |
| **魔法學徒** | 學會2種戲法和1種1環法術（對應職業列表，長休施放一次），施法關鍵屬性取決於所選職業 | 不推荐，收益较低，通常不如属性提升或專用专长 |
| **戰技專家** | 學會2種戰技，獲得1顆優勢骰（短休或長休回復） | 兼职战技大师且等级不高时可考虑，否则骰子太少 |
| **中甲大師** | 中甲不為隱匿檢定加劣勢，敏捷調整值獲得的AC加值變為+3 | 適合中甲潛行流派 |
| **靈活移動** | 移動速度增加，疾走時劣勢地形不降速，發動近戰攻擊後移動不招致借機 | **遊蕩者必點**，其他職業根據需要 |
| **中甲運用** | 獲得中甲和盾牌熟練項，力量或敏捷+1（上限20） | 需要中甲但职业不提供时才考虑 |
| **表演者** | 獲得樂器熟練項，魅力+1（上限20） | 吟游诗人自带，其他职业除非特别需要乐器 |
| **長柄武器大師** | 用特定長柄武器攻擊時，可用附贈動作進行握柄攻擊；目標進入範圍可發動借機攻擊 | 长柄武器流可选，注意某些效果可能有Bug |
| **堅如磐石** | 一項屬性+1（上限20），並獲得該屬性豁免檢定的熟練項 | 缺特定豁免熟练项时可考虑，但一般优先度不高 |
| **儀式施法者** | 學會2種任選的儀式法術 | 不推荐，仪式法术可通过其他方式获得 |
| **凶蠻打手** | 近戰武器攻擊時，**傷害骰擲兩次取高** | **近戰輸出強力推薦**，稳定提升伤害 |
| **神射手** | **遠程武器攻擊無視高地規則減值**；可選-5命中+10傷害 | **远程物理输出核心专长**，需注意命中率 |
| **盾牌大師** | 持盾時敏捷豁免+2；可反應法術敏捷豁免成功免傷，失敗減半 | **持盾前排推薦**，显著提升法術生存能力 |
| **熟習** | 選擇任意三種技能並獲得其熟練項 | 缺关键技能时考虑 |
| **法術狙擊** | 學會一種戲法，**觸發重擊所需攻擊骰點數-1**（可疊加） | 法系输出可选，注意效果是否正常触发 |
| **酒館毆鬥者** | 力量或敏捷+1（上限20），徒手、臨時武器或投擲物時，**傷害和攻擊骰可加雙倍力量調整值** | **投掷流核心**，但需注意版本变动 |
| **哨兵** | 近戰範圍內敵人攻擊盟友時，可用反應對該敵人攻擊；借機攻擊有優勢，且命中後目標本回合無法移動 | **控制型近戰推薦**，保护队友，限制敌人移动 |
| **武器大師** | 選擇四種武器獲得熟練項，力量或敏捷+1（上限20） | 缺特定武器熟练项时考虑，近战职业通常不缺 |
| **戰地施法者** | 維持法術專注的豁免檢定有優勢；可用反應對移出近戰範圍的目標施展電爪 | 前排法師或需要維持專注的施法者可考慮 |
| **健壯** | **每級生命上限+2**（計算基於當前等級） | **通用生存向專長**，任何角色都可以考慮，提升血量 |
