攻击骰
#

非重击命中伤害期望
#

伤害公式 = 伤害骰数量n * 伤害骰面数d + 伤害加值Bonus_damage(Bd)
伤害期望Dp = n * (1+d)/2 + Bd

攻击骰1d20计算优劣势
#

  1. 掷20-(重击减值)：必定命中且重击
  2. 掷1：严重失误，必定未命中
  3. 掷2~(20-重击减值-1)：假设攻击骰掷x(2 <= x <= 20-重击减值-1)
    命中条件：x + 攻击加值 >= 敌方护甲AC

D20掷骰结果为k的概率Pk
#

k∈[1,20]

  1. 均势: Pk = 1/20
  2. 优势: P(max=k) = (2k-1)/400
  3. 劣势: P(min=k) = (41-2k)/400

重击命中率Ph = ∑k=20-重击减值20Pk
大失败率Pf = Pk=1
非重击命中率Pn = ∑k=(Ac-Bonus)20-重击减值-1Pk

攻击加值
#

  1. 攻击属性调整值：
    1. 近战非灵巧武器 & 灵巧武器但是力量>敏捷：使用力调
    2. 灵巧武器且敏捷>力量 & 远程武器： 使用敏调
    3. 攻击骰法术：使用施法主属性调整值
  2. 对应攻击属性熟练项加值（固定存在，总等级1~4级+2，5~8级+3，9~12级+4）
  3. 增益buff
  4. 装备加值

攻击骰实际伤害期望
#

实际伤害期望Dexp = 非重击命中伤害期望Dp * (重击命中率Ph * 2 + 非重击命中率Pn)
实际命中率 = (重击命中率 * 2 + 非重击命中率)
= Ph * 2 + Pn
= ∑k=20-重击减值20Pk * 2 + ∑k=(Ac-Bonus)20-重击减值-1Pk

四元函数形式，公式基于投骰方式（均势、优势、劣势）和参数：

  • 非重击命中伤害期望 (D_p)
  • 敌方护甲 AC
  • 攻击加值 Bonus
  • 重击减值 C

其中，令 (B = AC - Bonus)，表示命中所需的最小骰子值（调整前）。

简化后的伤害期望公式
公式假设 (B) 在 1 到 20 之间，如果 (B < 1)，则设 (B = 1)。

1. 均势
#

\[ D_{exp} = D_p \times \frac{2(C+1) + \max(20-C-B, 0)}{20} \]

2. 优势
#

\[ D_{exp} = D_p \times \frac{2(1+C)(39-C) + \max(20-C-B, 0) \times (B + 18 - C)}{400} \]

3. 劣势
#

\[ D_{exp} = D_p \times \frac{2(1+C)^2 + \max(20-C-B, 0) \times (22 - B + C)}{400} \]

豁免骰
#

豁免检定骰1d20计算优劣势
#

  1. 掷20：必定豁免成功
  2. 掷1：必定豁免失败
  3. 掷2~19：假设豁免检定骰掷x
    1. 豁免AC = x + 法术豁免属性 + 豁免属性熟练项加值（0-没有熟练项 或者 总等级1~4级+2，5~8级+3，9~12级+4-拥有熟练项） + 装备豁免检定AC加值
    2. 豁免失败条件：豁免DC >= 豁免AC

豁免DC
#

  1. 非固定豁免DC
    豁免DC = 8 + 施法属性调整值 + 施法属性熟练项加值（固定存在，总等级1~4级+2，5~8级+3，9~12级+4） + 装备法术豁免等级DC加值
  2. 固定豁免DC：捏豁免法术卷轴

技能伤害期望
#

  • 敌方AC=15
  • 重击减值C=0
  • 力调=3，等级=4（熟练项加值=2） -> 攻击加值Bonus = 5
    D20掷骰结果为k的概率Pk, k∈[1,20]
  1. 均势: Pk = 1/20
  2. 优势: P(max=k) = (2k-1)/400
  3. 劣势: P(min=k) = (41-2k)/400

巨武器大师
#

专长减值dim = 5
伤害加值Db = 10
例：3力调至上真神之视，双手1d8，伤害骰1d8 + 3 -> 非重击伤害期望=7.5

实际命中率：(重击命中率Ph * 2 + 非重击命中率Pn)

重击命中率Ph = Pk=20 =

  • 均势: 1/20
  • 优势: 0.0975
  • 劣势: 0.0025

关闭巨武器大师
#

  • 非重击伤害期望=7.5
  • 非重击命中率Pn： ∑k=(Ac-Bonus)20-C-1Pk = ∑k=1019Pk
  • 均势: 0.5 -> 3.75
  • 优势: 0.7 -> 5.25
  • 劣势: 0.3 -> 2.25

打开巨武器大师
#

  • 非重击伤害期望=17.5
  • 非重击命中率Pn： ∑k=(Ac-Bonus+dim)20-C-1Pk = ∑k=1519Pk
  • 均势: 0.25 -> 4.375
  • 优势: 0.4125 -> 7.21875
  • 劣势: 0.0875 -> 1.53125
