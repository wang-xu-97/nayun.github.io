日志关键字处理
#

需求：

  1. 1次遍历，O(n)
  2. 多关键字(支持regex pattern)
  3. 关键字一对一自定义处理方法解耦
  4. 处理结果统计，每个关键字的成功/失败处理量+处理总量

例子：
找到日志中包含kwA的日志，执行dealA(line) -> pass/fail处理方法
找到日志中包含kwB的日志，执行dealB(line) -> pass/fail处理方法
…

思路：

  1. 每行日志查询全部关键字，有关键字出现，进入deallog回调
  2. 通过deal_log回调函数，在回调内部区分kw，执行不同逻辑
  3. 结束时输出关键字相关处理总量
</> python
 1import re
 2from collection import defaultdict
 3def keywords_check(checktitle, text, checklist, deal_log=(lambda pt, line:True)) -> int, str:
 4    # checklist 入参多类型支持(dict, list)
 5    checklist = list(dict.fromkeys(checklist))
 6    # 计数器
 7    kw_counter, err_counter = defaultdict(int), defaultdict(int)
 8    for line in text:
 9        found = [pattern for pattern in checklist if re.search(pattern, line) is not None]
10        if not found:continue
11        for pattern in found:
12            kw_counter[pattern] += 1
13            if False == deal_log(pattern, line):
14                err_counter[pattern] += 1
15    msg = "{} done, total {} records, {} errors\n{}".format(
16        checktitle
17        , (kw_sum:=sum(kw_counter.values()))
18        , sum(err_counter.values())
19        , "".join(["[{}]: {} records, {} errors\n".format(
20            pattern
21            , kw_counter.get(pattern, 0)
22            , err_counter.get(pattern, 0)
23            ) for pattern in checklist
24        ])
25    )
26    return kw_sum, msg
27
28checklist = {
29    'pt1':r"^kwA*$",
30    'pt2':r"^kwB*$",
31    'pt3':r"^kwC*$",
32}
33def deal_log(pt, line):
34    if pt == checklist['pt1']:
35        return dealA(line)
36    elif pt == checklist['pt2']:
37        return dealB(line)
38    elif pt == checklist['pt3']:
39        return dealC(line)
40    else:
41        return False
42
43keywords_check('check keywords', checklist.values(), deal_log)

使用linux grep管道
#

优点：

  • 极高性能：grep 用 C 编写，高度优化（如 Boyer-Moore 算法）。
  • 并行处理：部分 grep 实现支持多线程（如 GNU grep 的 –parallel）。
  • 管道优化：内核级 I/O 缓冲减少磁盘访问次数。

缺点：

  • 不支持正则表达式组合
  • 不支持逐行后处理
</> python
1import subprocess
2subprocess.run(["grep", "-F", "keyword", "large.log"])
